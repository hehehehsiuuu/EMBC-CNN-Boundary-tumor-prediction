# EMBC-CNN-Boundary-tumor-prediction

Learning-based CNN boundary prediction for brain tumor MRI with **leak-free (patient-level) evaluation**.  
This repository accompanies our EMBC 2026 paper draft: *Learning-Based CNN Boundary Prediction with Leak-Free Evaluation in Brain Tumor MRI*.

## Paper
- Title: Learning-Based CNN Boundary Prediction with Leak-Free Evaluation in Brain Tumor MRI
- Authors: Vikranth Singh, Soumyasanta Laha
- Status:  [under review]
